# Glossary

- **RIFCC-DA**: Role, Input, Format, Constraints, Checks, and Data-specific assumptions.
- **Sentinel value**: Placeholder code representing missing/invalid data (for example 999, -1, 9999).
- **PII-adjacent**: Non-direct identifiers that still require restricted handling.
- **Row reconciliation**: Before/after row counts plus counts removed at each cleaning step.
- **EDA**: Exploratory Data Analysis used to answer business questions with evidence.
- **Self-contained dashboard**: One HTML file with embedded Plotly JS and all scenario charts.
- **Scenario letter**: `A`, `B`, or `C`, used for artifact filenames.
